# Flydigi-Vader-5S

Fix Flydigi-Vader-5S not working on Linux.

1. `sudo cp 99-flydigi-vader-5s.rules /etc/udev/rules.d/`.
2. `reboot`
3. Unplug and plug the game controller twice, it should work.

-----

修复飞智黑武士 5S 在 Linux 上无法使用。

2. `reboot`
3. 拔插两次游戏手柄，应该就能正常使用了。
